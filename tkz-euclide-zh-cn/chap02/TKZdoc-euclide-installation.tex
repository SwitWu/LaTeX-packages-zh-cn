\section{安装}

% \tkzNamePack{tkz-euclide} and \tkzNamePack{tkz-base} are now on the server of
% the \tkzname{CTAN}\footnote{\tkzNamePack{tkz-base} and \tkzNamePack{tkz-euclide}
% are part of \NameDist{TeXLive} and \tkzname{tlmgr} allows you to install them.
% These packages are also part of \NameDist{MiKTeX} under \NameSys{Windows}.}. If
% you want to test a beta version, just put the following files in a texmf folder
% that your system can find.
% You will have to check several points:
\tkzNamePack{tkz-euclide}和\tkzNamePack{tkz-base}现在已被收录于
\tkzname{CTAN}\footnote{\tkzNamePack{tkz-base}和\tkzNamePack{tkz-euclide}
已被\NameDist{TeXLive}发行版收录，可通过\tkzname{tlmgr}安装。
在\NameSys{Windows}平台，\NameDist{MiKTeX}发行版也收录了这两个宏包。}。
如果需要测试开发版，则需要按texmf的目录结构将必要的文件复制到指定目录。
并注意以下几点：

% \begin{itemize}\setlength{\itemsep}{5pt}
% \item The \tkzNamePack{tkz-base} and \tkzNamePack{tkz-euclide} folders must be
% located on a path recognized by \tkzname{latex}.
% \item The \tkzNamePack{xfp}\footnote{\tkzNamePack{xfp} replaces
% \tkzNamePack{fp}.}, \tkzNamePack{numprint} and \tkzNamePack{tikz 3.00} must be
% installed as they are mandatory, for the proper functioning of
% \tkzNamePack{tkz-euclide}.
% \item This documentation and all examples were obtained with
% \tkzname{lualatex-dev} but \tkzname{pdflatex} should be suitable.
% \end{itemize}
\begin{itemize}\setlength{\itemsep}{5pt}
\item \LaTeX{}必须能够识别\tkzNamePack{tkz-base}和\tkzNamePack{tkz-euclide}目录。
\item 必须正确安装\tkzNamePack{xfp}\footnote{\tkzNamePack{xfp}替代了
\tkzNamePack{fp}。}、 \tkzNamePack{numprint}和\tkzNamePack{tikz 3.00}及以上版本才能够正常使用
\tkzNamePack{tkz-euclide}.
\item 该说明文档和所有的示例都是使用\tkzname{lualatex-dev}编译的，
但使用\tkzname{pdflatex}和\tkzname{xelatex}也能够正常编译。
\end{itemize}

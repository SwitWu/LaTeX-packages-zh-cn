# LaTeX-packages-zh-cn
LaTeX常用宏包中文翻译版
> 注意: 源文件只存放所有编译所需的`.tex`、`.sty`和`.cls`等文件, 一般不包含相应的`pdf`文件, 我将所有最终输出的`pdf`文件都放在releases之中方便进行下载.

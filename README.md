# LaTeX-packages-zh-cn


## 介绍

本仓库用于收集 $\mathsf{\LaTeX}$ 常用宏包手册的中文翻译版本。

欢迎感兴趣的朋友提交 Issues 以及 pull requests，提升自己的翻译水平和 $\mathsf{\LaTeX}$ 技术水平，同时也是为社区做贡献。

## 如何获取 PDF

后续可能不会在 Releases 中提供编译好的 PDF 文件了，大家直接下载源代码自行编译即可。
